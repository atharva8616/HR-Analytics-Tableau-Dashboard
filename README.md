# HR Analytics Dashboard (Tableau Project)

## Project Overview
This project focuses on analyzing HR employee data to identify workforce trends, employee attrition patterns, and key factors affecting employee retention. The dashboard was built using Tableau to provide interactive insights for HR decision-making.

## Objectives
- Analyze employee attrition rate
- Identify departments with highest turnover
- Understand employee demographics
- Evaluate salary and experience impact on attrition

## Tools Used
- Tableau
- Excel / CSV Dataset
- Data Cleaning
- Data Visualization

## Dataset Description
The dataset contains employee information including:
- Age
- Gender
- Department
- Job Role
- Salary
- Attrition Status
- Years at Company

## Dashboard Features
The dashboard provides the following visualizations:

1. Attrition Rate Overview
2. Department-wise Employee Distribution
3. Gender Distribution
4. Salary vs Attrition Analysis
5. Experience vs Employee Retention

## Key Insights
- Higher attrition observed in sales department
- Employees with lower experience show higher turnover
- Salary bands influence employee retention
- Majority of employees belong to mid-age workforce

# Business Insights

1. Sales department has the highest attrition rate compared to other departments.
2. Employees with less than 3 years of experience show higher turnover.
3. Salary bands below average show higher employee exits.
4. Mid-career professionals (age 28-35) form the largest workforce segment.

Recommendations:
- Improve employee engagement in sales department
- Provide better salary incentives for early career employees
- Introduce retention programs for high-risk employees
